# A Photo Sharing App
## Description 
Instagrammy is a picture sharing android app that allows users to register and then take and upload pictures with a description and hashtags, some of which are autogenerated. These pictures are shared with other users of the app. The users can view the photo in full size by clicking on the thumbnail and can also post comments on those images.
